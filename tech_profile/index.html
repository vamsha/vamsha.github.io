RGen – Natural Language to SQL Engine (2019)

Built an internal tool, RGen, to convert natural language queries into SQL and execute them across multiple databases via custom connectors. Developed in 2019—well before LLMs and AI agents were mainstream—RGen used rule-based NLP and intent mapping to enable non-technical users to query data conversationally. Designed for compatibility with MySQL, Oracle, and PostgreSQL, it reduced dependency on technical teams and improved data accessibility. The tool demonstrated early innovation in AI-driven data interaction and laid the groundwork for later conversational analytics efforts.
